V2::Mutant API

Level 2: This will check if the adn is mutant or human

Check if adn is human

POST /v2/mutant

When the adn is human it will return 403

Parameters

Name Description
dna DNA it is a square matrix from 4x4 to 12x12. Each element can only contain zero or more of the valid characters: (A,T,C,G)

Request

Headers

Content-Type: application/json
Host: example.org
Cookie:

Route

POST /v2/mutant

Body

{"dna":["ATGCGA","CAGTGC","TTATTT","AGACGG","GCGTCA","TCACTG"]}

cURL

curl "https://dna-bank-205308.appspot.com/v2/mutant" -d '{"dna":["ATGCGA","CAGTGC","TTATTT","AGACGG","GCGTCA","TCACTG"]}' -X POST \
	-H "Content-Type: application/json" \
	-H "Host: example.org" \
	-H "Cookie: "

Response

Headers

Content-Type: text/html
Cache-Control: no-cache
X-Request-Id: bc9cc1e1-8064-4958-9530-ef6af31ebe31
X-Runtime: 0.001809
Content-Length: 0

Status

403 Forbidden
